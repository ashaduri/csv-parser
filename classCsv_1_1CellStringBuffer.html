template<std::size_t Size>

A helper class for compile-time buffer construction and string unescaping.

Template Parameters

Constructor.

Copies the cell data into the buffer. If the cell data size is greater than the buffer size, the buffer is marked as invalid and isValid() will return false.

Parameters
cellCell data to copy into the buffer.
has_escaped_quotesIf true, the data in cell is cleaned up (unescaped) before being copied into the buffer.
