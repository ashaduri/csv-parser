Detailed Description

template<typename CellT>
class Csv::CellTrait< CellT >

Parser::parse*() functions use this to create the an element of a container.

By default, it handles Cell* classes (CellReference, etc.) and primitive numeric types (long int, double, etc.). This trait can be specialized for user-defined types.

Template Parameters
CellTType of the cell object to create.
